# xiang.js

xiang.js is a Javascript Xiangqi library with a simple AI.

### To run
```
% node shell.js
> version
0.0.3
> load games/许银川让九子对聂棋圣.pgn
75 moves
> ascii

｜砲・・・・・砲｜

Bye!
```
